# 🛠️ Skills 开发脑图

## 中心主题：Skills 开发

### 1️⃣ Skill 结构
```
Skill 包
├── SKILL.md (描述文件)
│   ├── <skill> 标签
│   ├── name (名称)
│   ├── description (描述)
│   └── location (路径)
├── index.js (主逻辑)
│   ├── 导入依赖
│   ├── 辅助函数
│   ├── 导出函数
│   └── 错误处理
├── package.json (依赖配置)
│   ├── name
│   ├── version
│   └── dependencies
└── assets/ (资源文件)
    ├── 图片
    ├── 模板
    └── 配置文件
```

### 2️⃣ 开发流程
```
1. 需求分析
   ├── 功能定义
   ├── 输入输出
   └── 边界情况

2. 编写代码
   ├── 创建文件结构
   ├── 实现主逻辑
   ├── 添加错误处理
   └── 编写注释

3. 测试调试
   ├── 单元测试
   ├── 集成测试
   └── 日志输出

4. 注册配置
   ├── 添加到 config.json
   ├── 设置启用状态
   └── 配置参数

5. 文档编写
   ├── 使用说明
   ├── 示例代码
   └── 常见问题
```
