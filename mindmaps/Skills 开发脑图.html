Skills 开发全景图

8 大核心模块，掌握 Skills 开发的完整技能树

📦 Skill 结构

  • SKILL.html - 技能描述、触发规则、使用说明
  • index.js - 主逻辑代码、工具调用、业务实现
  • package.json - 依赖管理、版本信息、脚本命令
  • README.html - 详细文档、使用示例、API 说明
  • assets/ - 资源文件（图片、模板等）
  • tests/ - 单元测试、集成测试

🔄 开发流程

  • 需求分析 - 明确目标用户、使用场景、核心功能
  • 设计阶段 - 架构设计、接口设计、数据模型
  • 编码实现 - 编写代码、调用工具、错误处理
  • 调试测试 - 日志输出、断点调试、单元测试
  • 文档编写 - SKILL.html、README、使用示例
  • 发布分享 - 代码整理、版本标记、社区分享

⚡ 核心能力

  • 文件操作 - read、write、edit、exec
  • 网络请求 - web_fetch、browser、API 调用
  • 数据处理 - JSON、CSV、Excel、PDF
  • 外部 API - 天气、股票、翻译、AI 服务
  • 定时任务 - cron、every、at 调度
  • 消息通知 - message、broadcast、提醒

🐛 调试技巧

  • 日志输出 - console.log、debug 信息
  • 错误处理 - try-catch、错误日志、用户提示
  • 断点调试 - debugger、vscode 调试配置
  • 性能分析 - 执行时间、内存占用、优化点
  • 测试用例 - 边界条件、异常场景、回归测试
  • 用户反馈 - 收集问题、快速迭代、持续优化

🏆 最佳实践

  • 代码规范 - ESLint、Prettier、注释清晰
  • 错误处理 - 友好的错误提示、降级方案
  • 性能优化 - 缓存、懒加载、异步处理
  • 文档编写 - 清晰的说明、完整的示例
  • 版本管理 - Git、语义化版本、更新日志
  • 用户友好 - 简单的配置、清晰的提示

📚 示例 Skills

  • quote-of-day - 每日一句（入门级）
  • todo-list - 待办事项管理（进阶级）
  • file-processor - 文件处理工具（进阶级）
  • web-scraper - 网页爬虫（高级）
  • weather - 天气查询（API 调用）
  • stock-watcher - 股票监控（数据可视化）

🔧 常用工具库

  • fs/path - Node.js 文件系统模块
  • execSync - 执行 shell 命令
  • axios - HTTP 请求库
  • cheerio - HTML 解析
  • puppeteer - 浏览器自动化
  • moment/dayjs - 日期时间处理

🚀 发布流程

  • 代码整理 - 清理冗余代码、优化结构
  • 文档编写 - README、使用说明、API 文档
  • 测试验证 - 完整测试、边界测试、性能测试
  • 版本标记 - git tag、版本号、更新日志
  • 社区分享 - GitHub、Discord、技术博客
  • 持续维护 - Bug 修复、功能迭代、用户支持

💡 学习路径

建议从quote-of-day入门，理解 Skill 基本结构。
然后尝试todo-list或file-processor，掌握文件操作和数据处理。
最后挑战web-scraper等高级 Skills，实现复杂功能。
