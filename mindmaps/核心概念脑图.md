# 🧠 OpenClaw 核心概念脑图

## 中心主题：OpenClaw

### 1️⃣ 架构组成
```
OpenClaw
├── Gateway (网关)
│   ├── 消息路由
│   ├── 渠道管理
│   └── 配置中心
├── Sessions (会话)
│   ├── Main Session (主会话)
│   └── Isolated Session (隔离会话)
├── Agents (智能体)
│   ├── 主 Agent
│   └── 子 Agent (spawn)
└── Skills (技能)
    ├── 内置技能
    └── 自定义技能
```

### 2️⃣ 消息流程
```
用户消息
    ↓
渠道接入 (Telegram/微信/Discord)
    ↓
Gateway 接收
    ↓
Session 处理
    ↓
Agent 思考 + 工具调用
    ↓
Skills 执行
    ↓
回复生成
    ↓
渠道发送
    ↓
用户收到
```

### 3️⃣ 记忆系统
```
MEMORY.md (长期记忆)
├── 重要决策
├── 用户偏好
└── 关键事件

memory/YYYY-MM-DD.md (日常记忆)
├── 对话日志
├── 临时笔记
└── 待办事项

memory_search (语义搜索)
└── 精准回忆
```

### 4️⃣ 定时任务
```
Cron Jobs
├── at (一次性)
├── every (周期性)
│   ├── 进度汇报
│   ├── 健康检查
│   └── 定期检查
└── cron (表达式)
    ├── 每日任务
    ├── 每周任务
    └── 自定义周期

Payload
├── systemEvent (主会话)
└── agentTurn (隔离会话)
```

### 5️⃣ Token 优化
```
提示词优化
├── 精简表达
├── 模板化
└── 示例驱动

上下文管理
├── 定期清理
├── 摘要压缩
└── 分层记忆

响应控制
├── 长度限制
├── 格式指定
└── 思维链开关

缓存复用
├── 问题缓存
├── 模板缓存
└── 结果缓存
```

### 6️⃣ Skills 开发
```
Skill 结构
├── SKILL.md (描述)
├── index.js (逻辑)
├── package.json (依赖)
└── assets/ (资源)

开发流程
├── 需求分析
├── 编写代码
├── 测试调试
└── 注册配置

最佳实践
├── 单一职责
├── 错误处理
├── 文档完善
└── 日志记录
```

### 7️⃣ 渠道集成
```
IM 渠道
├── Telegram
├── WhatsApp
├── Discord
├── Slack
└── 微信

文档渠道
├── Feishu (飞书)
├── Notion
└── Google Docs

存储渠道

```

