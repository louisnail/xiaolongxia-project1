# 📦 安装指南 - 15 分钟快速上手

> 零基础也能看懂的安装教程

## 🎯 前置要求

- Node.js 18+ 
- Git
- 一个 IM 账号（微信/Telegram/Discord 等）

## 📝 步骤 1: 克隆项目

```bash
# 克隆 OpenClaw 仓库
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 安装依赖
npm install
```

**预期输出**:
```
added 1234 packages in 45s
```

## ⚙️ 步骤 2: 基础配置

创建配置文件：

```bash
# 复制示例配置
cp config.example.json config.json
```

编辑 `config.json`：

```json
{
  "gateway": {
    "port": 3000
  },
  "channels": [
    {
      "type": "telegram",
      "token": "YOUR_BOT_TOKEN"
    }
  ]
}
```

> 💡 **提示**: 不同渠道的配置方式见 [配置详解](03-配置详解.md)

## 🚀 步骤 3: 启动服务

```bash
# 启动 Gateway
npm run gateway:start

# 或使用 PM2（推荐生产环境）
pm2 start npm --name "openclaw" -- run gateway:start
```

**验证启动**:
```bash
# 检查状态
curl http://localhost:3000/health

# 预期输出: {"status":"ok"}
```

## 🔗 步骤 4: 连接渠道

### Telegram
1. 找 @BotFather 创建机器人
2. 获取 Token 填入配置
3. 发送 `/start` 测试

### 微信
1. 使用 WeChaty 或企业微信
2. 扫码登录
3. 发送消息测试

### Discord
1. 创建 Discord 应用
2. 邀请机器人到服务器
3. 配置 Bot Token

## ✅ 验证清单

- [ ] Gateway 正常启动
- [ ] 端口可访问
- [ ] 渠道已连接
- [ ] 能收到消息
- [ ] 能发送消息

## 🐛 常见问题

### Q: 端口被占用？
```bash
# 查看占用端口的进程
lsof -i :3000

# 修改配置文件中的端口
```

### Q: 依赖安装失败？
```bash
# 清理缓存重试
npm cache clean --force
npm install

# 或使用 yarn
yarn install
```

### Q: 渠道连接不上？
- 检查 Token/密钥是否正确
- 确认网络可访问
- 查看日志：`tail -f logs/gateway.log`
