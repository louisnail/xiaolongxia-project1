⚙️ 配置详解

完全掌握配置文件，渠道接入，模型配置，环境变量

⏱️ 学习时长：1 小时  |  ⭐ 难度等级：中等

📁 配置文件结构

OpenClaw 的配置文件位于项目根目录的 config.yaml：

# config.yaml 完整示例 gateway: port: 3000 host: 0.0.0.0 logLevel: info channels: telegram: - token: "your-telegram-bot-token" name: "My Assistant" model: "qwen-plus" thinking: "on" systemPrompt: | 你是一位专业、友好、高效的 AI 助手。 你擅长用简洁清晰的语言解答问题。 models: qwen-plus: provider: dashscope apiKey: ${DASHSCOPE_API_KEY} model: qwen-plus maxTokens: 2000 memory: enabled: true path: ./MEMORY.md searchLimit: 5

🔑 环境变量配置

创建.env 文件存储敏感信息：

# .env 文件 DASHSCOPE_API_KEY=sk-your-key OPENAI_API_KEY=sk-your-key ANTHROPIC_API_KEY=sk-ant-your-key ELEVENLABS_API_KEY=your-tts-key

📱 渠道接入配置

Telegram

channels: telegram: - token: "bot-token-here" name: "OpenClaw Bot" model: "qwen-plus" thinking: "on" systemPrompt: "你是一位 AI 助手..."

微信

channels: wechat: - uin: "your-wechat-uin" name: "微信助手" model: "qwen-plus"

Discord

channels: discord: - token: "discord-bot-token" guildId: "your-server-id" model: "qwen-plus"

🤖 模型配置

通义千问（推荐）

models: qwen-plus: provider: dashscope apiKey: ${DASHSCOPE_API_KEY} model: qwen-plus maxTokens: 2000 temperature: 0.7

GPT-4

models: gpt-4: provider: openai apiKey: ${OPENAI_API_KEY} model: gpt-4-turbo maxTokens: 4000

Claude

models: claude-3: provider: anthropic apiKey: ${ANTHROPIC_API_KEY} model: claude-sonnet-4-20250514 maxTokens: 4000

🧠 记忆系统配置

memory: enabled: true path: ./MEMORY.md searchLimit: 5 extraPaths: - ./memory/*.md

⚡ 性能优化配置

performance: maxConcurrentSessions: 10 sessionTimeout: 1800000 # 30 分钟 cache: enabled: true ttl: 3600000 # 1 小时
