## 步骤 3: 理解基本概念

### Session (会话)
```
你 ↔️ OpenClaw 的每次对话 = 一个 Session

Main Session: 主对话（默认）
Isolated Session: 独立任务（后台运行）
```

### Agent (智能体)
```
Agent = AI 大脑

主 Agent: 处理日常对话
子 Agent: 专门任务（spawn 创建）
```

### Skill (技能)
```
Skill = 扩展能力

内置 Skills: 天气、股票、搜索...
自定义 Skills: 你自己开发的
```

## 步骤 5: 配置记忆系统

创建记忆文件：


## 步骤 6: 创建第一个定时任务

```bash
# 通过 API 创建
curl -X POST http://localhost:3000/cron \
  -H "Content-Type: application/json" \
  -d '{
    "action": "add",
    "job": {
      "name": "morning-greeting",
      "schedule": {
        "kind": "cron",
        "expr": "0 9 * * *"
      },
      "payload": {
        "kind": "systemEvent",
        "text": "☀️ 早上好！新的一天开始了"
      },
      "sessionTarget": "main"
    }
  }'
```

验证任务：
```
/list-cron
```

## 🐛 常见问题排查

### Q1: 没有回复
```
检查清单:
□ Gateway 是否运行
□ 渠道是否连接
□ 网络是否正常
□ 查看日志：tail -f logs/gateway.log
```

### Q2: 回复很慢
```
可能原因:
- 网络延迟
- 模型响应慢
- 任务复杂

解决:
- 检查网络
- 使用更快的模型
- 拆分复杂任务
```

### Q3: 技能不工作
```
检查:
□ Skill 是否正确注册
□ 路径是否正确
□ 依赖是否安装
□ 查看技能日志
```
