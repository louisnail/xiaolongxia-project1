🛠️ Skills 开发

从 0 到 1 创建自定义 Skills，扩展 AI 能力

⏱️ 学习时长：2 小时  |  ⭐ 难度等级：进阶

📦 Skill 结构

一个 Skill 包含以下核心文件：

  • SKILL.md - 技能描述和触发规则
  • index.js - 主逻辑代码
  • package.json - 依赖管理

📝 SKILL.md 模板

# skill-name ## Description 技能描述 ## Trigger 触发规则 ## Usage 使用示例

💻 index.js 模板

module.exports = { name: 'skill-name', trigger: /关键词/, async execute(context) { const { message, tools } = context; // 实现逻辑 return '回复内容'; } };

🔧 开发步骤

  1. 创建技能目录：mkdir skills/my-skill
  2. 编写 SKILL.md 描述文件
  3. 实现 index.js 逻辑
  4. 测试技能功能
  5. 部署到 skills 目录

📚 示例 Skills

  • quote-of-day - 每日一句（入门）
  • todo-list - 待办管理（进阶）
  • file-processor - 文件处理（进阶）
