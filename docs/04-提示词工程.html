💬 提示词工程

设计更好的提示词，获得更好的 AI 回复质量

⏱️ 学习时长：1 小时  |  ⭐ 难度等级：中等

🎯 什么是提示词

提示词（Prompt）是用户发送给 AI 的指令和上下文，决定了 AI 的回复质量和风格。

📝 提示词基本结构

# 好的提示词结构 角色定义 + 任务描述 + 约束条件 + 示例（可选）

🎭 角色扮演技巧

# 示例：专业助手 你是一位专业的 Python 开发工程师，擅长： 1. 编写清晰、可维护的代码 2. 遵循 PEP8 规范 3. 编写单元测试 4. 性能优化 请用专业的方式回答我的问题。

🔗 思维链（Chain of Thought）

# 示例：数学问题 请一步步思考并解答： 1. 理解问题 2. 列出已知条件 3. 选择合适的公式 4. 逐步计算 5. 验证答案 问题：小明有 5 个苹果，给了小红 2 个，又买了 3 个，现在有几个？

📚 少样本学习（Few-Shot）

# 示例：翻译任务 将中文翻译成英文： 例子 1: 中文：你好 英文：Hello 例子 2: 中文：谢谢 英文：Thank you 现在翻译： 中文：再见 英文：

⚡ 提示词优化技巧

1. 明确具体

# ❌ 模糊 帮我写代码 # ✅ 具体 用 Python 写一个函数，输入是列表，输出是列表中的最大值

2. 提供上下文

# ❌ 缺少上下文 这个怎么改？ # ✅ 有上下文 我在开发一个电商网站，用户登录功能有问题。 代码如下：[粘贴代码] 错误信息是：[粘贴错误]

3. 指定格式

请用 JSON 格式输出，包含以下字段： - name: 姓名 - age: 年龄 - city: 城市

🚀 高级技巧

1. 分步指令

请按以下步骤完成： 1. 分析问题 2. 提出 3 个解决方案 3. 对比优缺点 4. 推荐最佳方案

2. 自我反思

请回答后，再检查一遍答案： - 是否有逻辑错误？ - 是否有遗漏？ - 是否有更好的方案？

💡 实际应用示例

代码审查

你是一位资深代码审查工程师，请审查以下代码： 1. 找出潜在 Bug 2. 检查性能问题 3. 评估代码可读性 4. 提出改进建议 [粘贴代码]

文档编写

请为以下函数编写 API 文档，包含： - 功能描述 - 参数说明 - 返回值 - 使用示例 - 注意事项 [粘贴代码]
