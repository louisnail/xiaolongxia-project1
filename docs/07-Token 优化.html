💰 Token 优化

省钱就是赚钱，优化 Token 使用降低 30-60% 成本

⏱️ 学习时长：1 小时  |  ⭐ 难度等级：进阶

💡 Token 计费原理

API 调用按 Token 数量计费，包括输入和输出。

🎯 优化策略

1. 提示词精简

# ❌ 冗长 你是一位非常专业、友好、乐于助人、知识渊博的 AI 助手... # ✅ 精简 你是专业 AI 助手，简洁高效地回答问题。

效果：降低 30-50% 输入 Token

2. 上下文管理

# 限制历史消息数量 messageLimit: 10 # 只保留最近 10 条消息

效果：降低 40-60% 上下文 Token

3. 响应控制

# 使用 NO_REPLY 减少输出 当无需回复时，返回：NO_REPLY

效果：降低 20-30% 输出 Token

4. 缓存复用

# 避免重复计算 - 缓存常见问题答案 - 复用历史查询结果

5. 模型选择

# 简单任务用轻量模型 - 聊天：qwen-turbo（便宜快速） - 复杂任务：qwen-plus（能力强） - 代码：专用代码模型

📊 成本对比

优化前：月消费 ¥1000
优化后：月消费 ¥400-700
节省：30-60%
